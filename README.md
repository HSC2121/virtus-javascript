# Virtus JavaScript — Frontend Utilities & Features

A collection of JavaScript and frontend utilities powering **Virtus FE**. This repository houses reusable functionality, integrations, and UI helpers used across the Virtus frontend stack.

---

## 📦 Contents

- `index.html` — a sample or starting point to demo utilities  
- `js/` — directory with JavaScript modules, helpers, and integrations  
- (Optional future folders) `styles/`, `components/`, etc.

---

## 🚀 Features & Utilities

- Data formatting, time/date conversions, math / precision helpers  
- API wrappers or fetch utilities for backend endpoints  
- UI helper functions (DOM manipulation, event throttling, etc.)  
- Integration scaffolding for internal or third-party services
